# Adhan Audio Files

Place your adhan audio files (MP3 format) in this folder:

1. `adhan_makkah.mp3` - أذان مكة المكرمة
2. `adhan_madinah.mp3` - أذان المدينة المنورة
3. `adhan_alaqsa.mp3` - أذان المسجد الأقصى
4. `adhan_mishary.mp3` - أذان مشاري العفاسي

## Where to get Adhan audio files:

You can download free adhan audio files from:

- https://islamcan.com/audio/adhan/
- https://www.mp3quran.net/
- https://islamway.net/

Note: Make sure the audio files are royalty-free or properly licensed.
